# Hyprglass V1.5 production review

## Scope

V1.5 closes the remaining production boot and display-safety gaps before installing Hyprglass on a real laptop.

## Boot/session

- Added `greetd` and `greetd-tuigreet` to Arch and CachyOS package profiles.
- First setup configures a lightweight greetd login that starts Hyprland after authentication.
- Passwordless autologin is explicit through `--autologin`; it is not the default.
- Existing enabled display managers are not replaced unless `HYPRGLASS_FORCE_GREETD=1` is set.
- greetd config is backed up before replacement.

## Display safety

- Normal Settings apply still avoids display rules.
- Display apply now changes only the scale field inside the Hyprglass managed block.
- Custom resolution, refresh rate, monitor name, and position inside the managed block are preserved.
- Unmarked manual monitor configs remain protected.

## Verification added

- greetd/tuigreet package profile checks.
- installer boot-login configuration checks.
- managed display scale flow test proving custom resolution is preserved when only scale changes.
