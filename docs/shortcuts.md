# Hyprglass shortcuts

| Shortcut | Action |
|---|---|
| Super + Enter | Kitty terminal |
| Super + Space | Fuzzel launcher |
| Super + E | Files |
| Super + I / Super + , | Hyprglass Settings |
| Super + W/B/M/A/D/X | Hyprglass Settings |
| Super + Shift + P | Power menu |
| Super + Shift + L | Lock |
| Super + Shift + S | Screenshot selection to clipboard |
| Super + Q | Close window |
| Super + F | Fullscreen |
| Super + T | Toggle floating |
| Super + H/J/K/L or arrows | Move focus |
| Super + Shift + H/J/K/L or arrows | Move window |
| Super + R | Resize mode |
