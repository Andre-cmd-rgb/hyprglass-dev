# Hyprglass shortcuts

## Applications
| Shortcut | Action |
|---|---|
| Super+Enter | Open Kitty terminal |
| Super+Space | Open fuzzel launcher |
| Super+E | Open Nautilus file manager |
| Super+Shift+L | Lock with hyprlock |
| Super+Shift+S | Area screenshot → clipboard |

## Windows
| Shortcut | Action |
|---|---|
| Super+Q | Close active window |
| Super+F | Toggle fullscreen |
| Super+T | Toggle floating |
| Super+P | Toggle pseudotile (dwindle) |

## Focus & movement
| Shortcut | Action |
|---|---|
| Super+Arrow / Super+H/J/K/L | Move focus |
| Super+Shift+Arrow / Super+Shift+H/J/K/L | Move window |
| Super+R → Arrow/H/J/K/L | Resize mode (Escape or Enter to exit) |

## Workspaces
| Shortcut | Action |
|---|---|
| Super+1…9 | Switch workspace |
| Super+Shift+1…9 | Move window to workspace |

## Media & hardware
| Shortcut | Action |
|---|---|
| XF86AudioRaiseVolume / Super+F11 | Volume up |
| XF86AudioLowerVolume / Super+F10 | Volume down |
| XF86AudioMute / Super+F9 | Toggle mute |
| XF86AudioMicMute | Toggle mic mute |
| XF86MonBrightnessUp / Super+F6 | Brightness up |
| XF86MonBrightnessDown / Super+F5 | Brightness down |
| XF86AudioPlay / Super+F8 | Play/pause |
| XF86AudioPrev / Super+F7 | Previous track |
| XF86AudioNext | Next track |
