# Hyprglass design standard

Hyprglass is calm, dark, rounded and glassy. It borrows discipline from macOS spacing, not macOS identity.

Radius: bar 18px, window 14px, launcher 16px, notifications 14px. Gaps: inner 6, outer 10-14, top bar 8px inset. Border: active violet/blue, inactive dark blue-gray, 1-2px. Blur: subtle and readable. Shadows: soft, low opacity. Animation: fast and smooth, never circus-like. Typography: JetBrainsMono Nerd Font when present, monospace fallback.

Palette: base #070b1a, surface rgba(13,18,38,.68), strong surface rgba(17,24,48,.82), border rgba(129,140,248,.55), active border rgba(192,132,252,.90), text #dbeafe, muted #94a3b8, purple #c084fc, blue #60a5fa, pink #f0abfc, success #4ade80, warning #facc15, danger #fb7185.
